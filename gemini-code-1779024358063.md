# MINE ALBION COIN (ALC) - Telegram WebApp Clicker

Простой и стильный кликер для Telegram Mini Apps, стилизованный под экосистему TON.

## 🚀 Особенности
- **Адаптивный интерфейс:** Идеально отображается на любых смартфонах внутри Telegram.
- **Интеграция с Telegram WebApp API:** Автоматически подтягивает имя игрока и использует тактильную отдачу (Haptic Feedback) при кликах.
- **Мультитач:** Поддерживает клики несколькими пальцами одновременно.
- **Сохранение прогресса:** Баланс игрока сохраняется в `localStorage`.

## 🛠 Технологии
- HTML5 / CSS3 (Анимации, CSS переменные)
- Чистый JavaScript (Vanilla JS)
- Telegram Web App API

## 📦 Как запустить локально
Просто откройте файл `index.html` в любом современном браузере или запустите локальный сервер (например, через Live Server в VS Code).
